# Super-Mario
Това е моят опит да реализирам известната на всички ни игра - Супер Марио.

# Подготовка
-	За да се може да се стартира играта, изпълнете следните команди:

```
$ gem build Super-Mario
$ gem install Super-Mario-0.1.gem
```

- 	За да се стартира

	$ super_mario
