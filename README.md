# Super-Mario
Това е моят опит да реализирам известната на всички ни игра - Супер Марио.

# Подготовка
-	Изпълнете следните команди:

```
$ gem build Super-Mario.gemspec
$ gem install Super-Mario-0.1.gem
```

- 	За да се стартира:
```
$ super_mario
```
